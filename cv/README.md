# Dossier CV

Déposez ici vos CV au format PDF.

Noms recommandés :
- `cv-fr.pdf` pour la version française
- `cv-en.pdf` pour la version anglaise

Les liens de la page d’accueil pointent vers ces fichiers. Vous pouvez aussi n’en fournir qu’un seul ; dans ce cas, supprimez ou commentez le lien inutile dans `index.html`.
